`web-cli-guard`

## Short Description

Controlled web console for tmux-backed AI CLI tools with low-privilege runtime, audit logs, and elevated-command verification patterns.

## About Text

Web CLI Guard is a starter for teams that want a browser-based way to inspect or lightly operate existing AI CLI sessions while keeping the real execution boundary inside a low-privilege OS account, named tmux session, or sandbox.

It is useful for:

- viewing ongoing `codex`, `claude`, or shell-based AI work from the web
- giving staff a safer remote-operations surface than direct shell access
- adding approval or OTP checks before elevated commands
- combining web authentication, audit logs, and OS-level least privilege
